e-Clinic

E-Clinic is a digital healthcare platform designed to streamline online medical consultations between patients and healthcare providers. The system enables users to schedule appointments, consult with doctors virtually, and access medical records—all through a secure and user-friendly interface. Developed with both patients and clinic staff in mind, E-Clinic reduces the need for in-person visits, improves appointment management, and supports efficient patient care. The platform includes features for admin control, doctor scheduling, and patient history tracking, offering a comprehensive telemedicine solution.

WanderLift Traveling Company

WanderLift Online Booking is a user-friendly web application designed to streamline the reservation process for lift rides and adventure tours. Built to enhance convenience and reduce wait times, the platform allows users to browse available schedules, reserve slots, and receive real-time booking confirmations — all from any device. This project was created with a focus on responsive design, secure data handling, and a seamless user experience. It showcases my skills in front-end development, back-end integration, and user interface design. WanderLift also includes administrative tools for managing bookings and tracking capacity, making it ideal for tourism operators and outdoor recreation providers.
