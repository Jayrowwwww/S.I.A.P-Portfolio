e-Clinic

E-Clinic is a digital healthcare platform designed to streamline online medical consultations between patients and healthcare providers. The system enables users to schedule appointments, consult with doctors virtually, and access medical records—all through a secure and user-friendly interface. Developed with both patients and clinic staff in mind, E-Clinic reduces the need for in-person visits, improves appointment management, and supports efficient patient care. The platform includes features for admin control, doctor scheduling, and patient history tracking, offering a comprehensive telemedicine solution.

Baranggay Online Registration Website

A user-friendly web application designed to streamline the resident registration process for local barangays. This project allows residents to register online, submit required documents, and receive confirmation digitally, reducing manual paperwork and improving efficiency for barangay staff. Features include secure login, form validation, database integration, and an admin dashboard for managing submissions. Developed with responsiveness in mind to ensure accessibility across devices.

Hay Go Online Car Rental

A modern web-based car rental platform designed to simplify the process of booking vehicles online. The system allows users to browse available cars, make reservations, and manage their bookings with ease. It includes key features such as a responsive user interface, secure login system, real-time car availability, and an admin dashboard for managing rentals and customer data. Built with a focus on usability, efficiency, and a smooth user experience.
